# Idea scoring card — three axes + audit gate

> Each axis 1-5, total 15. **BlueOcean axis hard gate: cannot exceed 3 until it has passed the Phase 3 first-hand audit.**

## Three axes

### Demand (higher is better) — how painful, how frequent, how willing to pay
Evidence = real complaint posts + willingness-to-pay signals (NOT a tool's "opportunity score").
| 5 | High-frequency must-have; users actively seek alternatives / will pay; strong emotion |
| 4 | Clear pain, willingness to pay, high discussion heat |
| 3 | Real but "nice to have", lukewarm willingness to pay |
| 2 | Weak / niche / occasional need |
| 1 | Fake / self-indulgent need |

### BlueOcean (= 6 − measured competition intensity, **requires first-hand audit**)
| 5 | Almost nobody does it, or those who do solve it badly (clear gap) |
| 4 | Few players, generally poor UX, clear entry seam |
| 3 | Several decent competitors, still some differentiation room ← **UNAUDITED items cap here** |
| 2 | Red ocean, leaders entrenched, hard to differentiate |
| 1 | Giants dominate + network effects / capital steamroll / free-infra sweep; head-on = suicide |

### Feasibility (= 6 − launch difficulty, solo-dev lens)
| 5 | Weeks to MVP, off-the-shelf assembly, no heavy compliance/data moat |
| 4 | 1-2 months MVP, clear tech, manageable cold start |
| 3 | Months; some tech/data/compliance challenge |
| 2 | Heavy tech/data/compliance (medical/financial licensing), or structurally hard retention/monetization |
| 1 | Needs huge capital / licenses / proprietary data |

## Three recurring "blue-ocean killers" (audit specifically for these hidden competitors)
1. **Free-infrastructure sweep**: a free API / platform-native built-in already satisfies the core need (C-side willingness-to-pay ≈ 0).
2. **Platform API legal wall**: the selling point depends on aggregating someone's platform data, but the target platform has no API or its ToS bans aggregation/copying → cannot be built legally.
3. **Giant/incumbent already owns the core**: OS-level built-in or a brand already owns the core narrative; only the unpaid edges remain.
